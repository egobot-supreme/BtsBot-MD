# `🍓 Preguntas sobre la instalación 🍓`
<details>
<summary><b>Preguntas sobre las opciones de termux</b></summary>

  - [x] **¿Como inicio el procesó en termux?**<br>
  _Al descargar e instalar termux, no tiene que abrir la app, lo primero que tiene que hacer es irse a ajustes del termux, y darle permiso de almacenamiento, y recien puede entrar en termux, para iniciar el bot, de lo contrario en eso, si no hace ese proceso, al Instalar el bot en termux no funcionará, por lo cual tendra error en otros comandos, o en el punto en donde termux se detiene o se cierre solo._<br>
  
  - [x] **¿Donde descargo el termux?**<br>
  _Puedes ir aqui en este apartado para descargar directamente termux [`🚀 Aquí 🚀`](https://f-droid.org/es/packages/com.termux/). Te dare dos termux en esta pagina, si por un caso inesperado el termux no funciona aun que se den los permisos, descargue en este apartado [`🍓 Aquí 🍓`](https://www.mediafire.com/file/190g4b86nhyof7b/com.termux_118.apk/file)._
  
  - [x] **¿Tengo el bot instalado pero que hago si se cierra la app de térmux y como prenderlo?**<br>
  _Es muy facil volver a prender el bot en termux, si por que apagaste el celu o se cierre solo y quiere volver a prenderlo, escriba este comando en termux `cd BtsBot-MD`, despues te aparecera un apartado en termux de color verde y escribes este siguiente comando `npm start`, y listo ya está prendido, o tambien puede escribirlo de esta manera `cd BtsBot-MD && npm start`_
  <details><summary>Ejemplo</summary><img src="https://i.imgur.com/6nqEr5g.jpeg"></details>
  
  - [x] **¿Como iniciar el BtsBot si lo descargue por mediante una carpeta?**<br>
  _Si usted descargo el bot para tenerlo en su celular como carpeta, puede usar `npm start` para iniciar con el bot rápidamente._
  <details><summary>Ejemplo</summary><img src="https://i.imgur.com/vqxz6Lc.jpeg"></details>
</details>
